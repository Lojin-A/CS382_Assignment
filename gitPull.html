Staying Up-to-Date

The Git Pull & Merge Workflow

Lojin Ayesh

Update Main

1. Switch to Main: Leave your current branch.

git checkout main

2. Pull Changes: Download updates from GitHub.

git pull

Merge Updates

3. Return to your Branch: Go back to your work.

git checkout your-branch-name

4. Merge Main: Apply the updates to your code.

git merge main

Workflow Summary

Recap of the commands to update your environment:

git checkout main
git pull
git checkout your-branch-name
git merge main
